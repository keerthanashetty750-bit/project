# Retina Scan Offline Payment App

## Problem
Digital payments require internet connectivity which is unavailable in rural or disaster areas.

## Solution
An offline payment system using retina/eye scan authentication.

## Features
- Retina authentication
- Offline transaction storage
- Secure wallet
- Sync when internet available

## Tech Stack
Android Studio
Java
Python
TensorFlow
SQLite

## How it Works
1. User scans eye
2. AI verifies identity
3. Payment processed offline
4. Data synced later

## Future Improvements
- Real retina scanner hardware
- Blockchain transaction verification
